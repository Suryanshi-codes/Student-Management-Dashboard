📚 Student Management Dashboard

A simple Front-End Web Application to manage student data using HTML, CSS, and JavaScript (LocalStorage).
This project allows users to perform full CRUD operations (Create, Read, Update, Delete) in a clean and interactive UI.
🚀 Live Features
➕ Add new student details (Name, Course, Age)
📋 Display students in a styled table
✏️ Edit existing student data
❌ Delete student records
🔍 Search students by name or course
💾 Data stored in browser using localStorage
🎨 Modern dark UI with responsive design
🛠️ Tech Stack
HTML5
CSS3 (Dark UI Styling)
JavaScript (ES6)

📌 How It Works
➕ Add Student
Enter student name, course, and age → click Add Student
✏️ Edit Student
Click Edit button → update details via prompt → save automatically
❌ Delete Student
Click Delete button → removes student permanently
🔍 Search
Type name or course in search bar → table updates instantly
